## Version 0.1 (2025-01-25)

### 개발 배경
- 초등학교 3학년 아들(진성운)의 영어 단어 학습을 위해 제작
- 알파벳을 가끔 헷갈리는 수준의 초보 학습자 고려
- 2일에 한 번 꼴로 아빠와 함께 학습하는 시나리오

### 핵심 요구사항
1. **학습 대상**: 초등학교 3학년, 영어 초보 수준
2. **학습 목표**: 단어 스펠링 암기
3. **학습 방법**: 짝맞추기 + 스펠링 쓰기
4. **피드백**: 정답 시 발음 + 칭찬 효과음, 오답 시 재미있는 효과음
5. **학습 주기**: 신규 학습 + 복습의 2회 세션 구조
6. **다중 사용자**: 3명(아빠, 큰아들, 막내아들) 독립 학습

### 주요 기능

#### 1. 사용자 프로필 시스템
- 3명의 사용자 독립 관리
  - 👨‍💼 진호경 (파란색)
  - 👦 진성운 (초록색)
  - 🧒 진성율 (주황색)
- 사용자별 독립적인 단어장 데이터
- 사용자 전환 기능

#### 2. 단어 관리
- **이번 주 단어**: 현재 학습 중인 신규 단어
- **복습 단어**: 학습 완료 후 반복 학습용 단어
- 단어 추가/삭제 기능
- 발음 듣기 (Web Speech API)
- 단어장 간 이동 (이번 주 → 복습)

#### 3. 학습 모드

**짝맞추기**
- 한글 뜻 보고 4개 선택지에서 영어 단어 선택
- 최소 4개 단어 필요
- 틀리면 재도전 가능
- 넘어가기 버튼으로 어려운 단어 스킵

**스펠링 쓰기**
- 한글 뜻 보고 직접 영어 단어 입력
- 대소문자 구분 없음
- 발음 듣기 지원
- 틀리면 입력창 초기화 후 재도전
- 넘어가기 버튼으로 어려운 단어 스킵

#### 4. 피드백 시스템
- **정답**: 
  - 상승하는 화음 효과음
  - 단어 발음 재생
  - "🎉 잘했어요!", "🌟 완벽해요!" 메시지
- **오답**:
  - 하강하는 재미있는 효과음
  - "😊 다시 해볼까요!" 메시지
  - 재도전 또는 넘어가기 선택 가능

#### 5. 결과 화면
- 점수 및 정답률 표시
- 정답률별 격려 메시지
  - 100%: "🏆 완벽해요! 천재예요!"
  - 80%+: "🎖️ 정말 잘했어요!"
  - 60%+: "👍 좋아요! 조금만 더!"
  - 60% 미만: "💪 괜찮아요! 다시 해봐요!"

#### 6. CSV 내보내기/불러오기
- **내보내기**:
  - 파일명 형식: `단어장_[사용자명]_[날짜].csv`
  - 모든 단어(이번 주 + 복습) 포함
  - 엑셀에서 편집 가능
- **불러오기**:
  - CSV 파일 업로드
  - 기존 단어에 추가하는 방식
  - 대량 단어 입력 가능

### 기술 스택
- **프레임워크**: React
- **스타일링**: Tailwind CSS
- **아이콘**: Lucide React
- **데이터 저장**: localStorage (브라우저 로컬)
- **오디오**: Web Audio API, Web Speech API

### 개발 과정 주요 이슈 및 해결

**Issue 1: 틀렸을 때 진행 안됨**
- 문제: 오답 시 다음 문제로 넘어가지 않음
- 해결: 재도전 가능 + 넘어가기 버튼 추가

**Issue 2: 모바일 UI 프레임 이탈**
- 문제: 입력 칸이 좌우 배치로 화면 밖으로 나감
- 해결: 입력 칸을 위아래(세로) 배치로 변경

**Issue 3: 다중 사용자 관리**
- 문제: 여러 명이 사용할 때 데이터 분리 필요
- 해결: 사용자별 localStorage 키 분리

## 참고사항

### 브라우저 호환성
- Chrome, Safari, Edge (최신 버전)
- Web Speech API 지원 필요
- localStorage 지원 필요

### 제한사항
- 서버 없이 로컬 저장만 사용
- 기기 간 동기화 불가 (각 기기 독립)
- 브라우저 데이터 삭제 시 단어 소실 위험

### 권장 사용 방법
1. 정기적으로 CSV 백업
2. 각 아이에게 전용 기기 할당 (가능하면)
3. 학습 후 "복습으로 이동" 잊지 않기
