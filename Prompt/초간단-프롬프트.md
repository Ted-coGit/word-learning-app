# 🚀 초간단 버전 프롬프트

## 사용법:
1. 영어 단어 리스트 복사
2. 아래 프롬프트에 붙여넣기
3. Claude에게 보내기
4. 결과를 CSV 파일에 붙여넣기

---

## 프롬프트 (바로 복사해서 사용):

```
아래 영어 단어들을 초등학생용 영단어 학습 앱 CSV로 변환해주세요.

단어 리스트:
[여기에 영어 단어 붙여넣기]

형식: english,korean,partOfSpeech,phonetic,meanings,example1_en,example1_ko,example2_en,example2_ko,status

규칙:
- korean: 기본 뜻 한 단어
- meanings: 뜻 2-3개를 | 로 구분 (쉼표 금지!)
- 예문: 초등학생이 이해할 수 있는 쉬운 문장
- status: 모두 current
- 큰따옴표 사용 금지
- 헤더 출력 금지, 데이터만 출력

예시:
hello,안녕,감탄사,/həˈloʊ/,인사말|안녕하세요,Hello! How are you?,안녕하세요! 어떻게 지내세요?,Nice to meet you. Hello!,만나서 반가워요. 안녕!,current
```

---

## 💡 팁:

**한번에 10-20개 단어씩** 처리하는 게 좋습니다!

**단어만 입력 예시:**
```
apple
book
cat
dog
happy
```

**결과 예시:**
```
apple,사과,명사,/ˈæpəl/,사과|사과나무,I ate an apple.,나는 사과를 먹었어.,An apple a day keeps the doctor away.,하루 사과 하나면 의사가 필요 없어.,current
book,책,명사,/bʊk/,책|서적,I love this book!,나는 이 책을 정말 좋아해!,Please read this book.,이 책을 읽어주세요.,current
```
